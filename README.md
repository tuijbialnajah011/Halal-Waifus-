# Hwaifus

An elegant anime picture gallery providing a seamless experience to discover and download unlimited SFW and NSFW waifu, husbando, kitsune, and neko images. 

**Powered by 𝙱𝙹𝙴 ~ Clan**

## Features

- **Unlimited Discovery**: Browse a masonry grid of high-quality anime art.
- **SFW & NSFW Mode**: Readily toggle between safe-for-work and not-safe-for-work content.
- **Instant Downloads**: Save your favorite images straight to your device.
- **Modern & Responsive UI**: Smooth animations, dark mode theme with elegant typography.

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Framer Motion (for animations)
- Lucide React (for icons)
- Vercel (Ready for deployment)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

Enjoy the gallery!
